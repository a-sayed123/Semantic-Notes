## Download Attribute

**<h3 style="margin-block-end:0.5rem;">What is it?</h3>**
_This is an atribute in anchor tag notice the browser to save this file in the link in user downloads file that he specify in his browser settings_

### Types

<h3 style="margin-block-end:0.5rem; font-family: cursive;">Bolean</h3>

this will store the file in the href if the file in your site you can not download any file outside your site <br>
so if you put download in bolean shape it mean take the last part of the directory in href then store it with its name.
<br>

<h4 style="margin-block-end:0.5rem;">Contentable</h4>

This mean that you order the browser to store the file in the downloads folder with the name in 
the download attribute for example <br>

**</a href="E://animals/cats/cat.png" download></a>** <br>

*-- here when you click this link the browser will store this file with his native name in directory .* <br>

**</a href="E://animals/cats/cat.png" download="Big-cat.png"></a>** <br>

*-- here when you click this link the browser will store this file with the name in download attribute content .* <br>
